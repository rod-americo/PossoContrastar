# Kits cromáticos

Kits neutros de cor para adaptar a interface whitelabel sem carregar nomes,
logos, URLs ou assets de instituições. Eles preservam apenas direção cromática,
tokens básicos e um preview simples.

## Kits disponíveis

| Slug | Nome | Uso sugerido |
| --- | --- | --- |
| `noturno` | Noturno | Interfaces densas, operacionais e de baixa luminosidade |
| `botanico` | Botânico | Painel claro com verde institucional neutro |
| `lilás` | Lilás | Experiencias educativas, publicas ou de leitura leve |

## Contrato

- Não versionar marcas, logos, imagens, URLs institucionais ou nomes de origem.
- Cada kit deve manter `tokens.json`, `<slug>.css`, `preview.html` e `README.md`.
- Adaptadores podem mudar a apresentação, mas não devem alterar regras clínicas.
