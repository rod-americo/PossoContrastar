# Kit cromatico: Lilás

**Slug:** `lilas`

Tema claro com roxo, lilas e verde de apoio. Indicado para material educativo,
perguntas e respostas e experiencias de leitura mais abertas.

## Arquivos

- [`tokens.json`](tokens.json): tokens de cor, raio e tipografia.
- [`lilas.css`](lilas.css): variaveis CSS para integracao rapida.
- [`preview.html`](preview.html): preview estatico do kit.
