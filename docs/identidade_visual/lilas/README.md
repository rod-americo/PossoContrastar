# Kit cromático: Lilás

**Slug:** `lilas`

Tema claro com roxo, lilás e verde de apoio. Indicado para material educativo, perguntas e respostas e experiências de leitura mais abertas.

## Arquivos

- [`tokens.json`](tokens.json): tokens de cor, raio e tipografia.
- [`lilas.css`](lilas.css): variáveis CSS para integração rápida.
- [`preview.html`](preview.html): preview estático do kit.
