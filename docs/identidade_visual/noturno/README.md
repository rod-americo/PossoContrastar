# Kit cromático: Noturno

**Slug:** `noturno`

Tema escuro de alta densidade, com base azul-petróleo, ciano e semântica de
alerta bem visível. Indicado para painéis operacionais e ambientes de leitura
prolongada.

## Arquivos

- [`tokens.json`](tokens.json): tokens de cor, raio e tipografia.
- [`noturno.css`](noturno.css): variáveis CSS para integração rápida.
- [`preview.html`](preview.html): preview estático do kit.
