# 4. Precauções com pacientes em diálise de manutenção

| Tópico | Meio de contraste iodado | Meio de contraste à base de gadolínio (MCBG) |
| --- | --- | --- |
| **Riscos relacionados ao uso de contraste** | - Risco aumentado de IRA-AC para pacientes dialíticos não anúricos (seguir orientações para pacientes de risco para IRA-AC).<br>- Para pacientes dialíticos anúricos (diurese < 100 mL/dia), sem transplante funcional: podem receber meio de contraste iodado intravascular, sem risco de dano renal adicional. | - Risco aumentado de fibrose sistêmica nefrogênica (FSN) para dialíticos não anúricos e anúricos.<br>- É contraindicado o uso de MCBG do Grupo I. |
| **Programação de diálise** | - Não é necessário reprogramar a diálise após o uso do contraste; manter programação habitual das diálises do paciente. | - Não é necessário reprogramar a diálise após o uso do contraste (Grupo II), porém é recomendado agendar o exame para o mais próximo possível da próxima sessão de diálise.<br>- ESUR 2025: também recomenda diálise adicional dentro de 48 horas quando usar meio de contraste linear. |
